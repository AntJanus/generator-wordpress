# generator-wordpress

> Yeoman generator for [Genesis WordPress][1].


## Dependencies

- [Yeoman][2]

```
$ npm install -g yo
```


## Installation

```
$ npm install -g generator-genesis-wordpress
```

*If you get EMFILE issues, try running: `ulimit -n 4096`*


## Getting Started

**The full documentation can be found with [Genesis WordPress][1].**
